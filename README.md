## Overview

Welcome to the Web Application of the Mahinda Rajapaksha College, Homagama! This website is designed using HTML, CSS, JavaScript, and Bootstrap to showcase Glorious history, Cultural assets, Talented students, Academic goals, and Sports achievements of our Mahinda Rajapaksha College, Homagama. The web application offers an attractive and colorful interface to explore the journey of Mahindians since year 2011.

## Features

- **Home:** Explore services and opportunities provided by the school.
- **About Us:** Recognize the valuable assets that represents  the school pride.
- **History:** Browse the timeline to see achieved  goals through the journey. 
- **Events:** Enjoy the extra-curricular activities in an interactive gallery.
- **News:** View student achievements on academics, aesthetics, and sports.
- **Contact Us:** Feel free to contact school for inquiries or feedback.

## Technologies Used

- **Languages:** HTML, CSS, JavaScript
- **Framework:** Bootstrap

## Getting Started

To explore the Web Application, follow these steps:

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/Teshan2000/School-Website.git
   ```

2. **Open the Project:**
   - Navigate to the project directory.
   - Open the `home.html` file in your preferred web browser to view the website.

3. **Explore the Features:**
   - Browse through the Home page, explore the History and About Us pages of this wonderful academy and view the events and the news pages filled with memories of our talented students.
